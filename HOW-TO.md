# Prerequisites
1. node (v8.9.2)
2. npm (5.5.1)
NOTE: It should work with latest node version, but was written and tesed with node (v8.9.2) and npm (5.5.1)

# Build and run
1. Unpack zip
2. cd ./target365-node-sdk
3. npm install
4. npm test
5. Use target365-client.js

# Modify files
1. User any text editor to edit files
